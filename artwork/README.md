# CloudEvents logos


In the artwork repo, cloudevents logos are available in 3 formats (PNG/SVG/AI), 3 layouts -- horizontal (also known as landscape format), stacked (which is closer to square), and icon (which does not include the name and is square), and 3 versions (color/black/white). For questions about using terms in text, please see the CNCF Style Guide.
